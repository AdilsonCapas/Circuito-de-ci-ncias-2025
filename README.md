# 💧 Jogo Educativo de Análise de pH da Água
Projeto desenvolvido utilizando a Unity com foco em educação ambiental, voltado à análise da qualidade da água por meio de mecânicas interativas.
🎯 Objetivo
O jogo tem como objetivo ensinar, de forma prática e visual, a identificar níveis de pH e compreender quais são adequados para o consumo.
🎮 Mecânicas do jogo
🧪 Classificação de pH
O jogador interage com gotas que representam diferentes níveis de pH
Cada gota possui uma coloração baseada na tabela de pH
O objetivo é arrastar a gota para o balde correto:
✅ Ideal para consumo
❌ Não ideal
⏱️ Sistema de desafio
Tempo total: 60 segundos
Objetivo: acertar 10 classificações
Penalidade: a cada erro, o tempo é reduzido em 3 segundos
📷 Integração com câmera
Utilização da câmera do dispositivo
Aplicação prática após experimentos reais
Conexão entre o ambiente físico e digital
🛠️ Tecnologias utilizadas
C#
Unity
Desenvolvimento para dispositivos mobile
🧠 Aprendizados
Aplicação de lógica baseada em dados (tabela de pH)
Criação de sistemas interativos com feedback em tempo real
Desenvolvimento de minigames educativos
Integração entre recursos do dispositivo e gameplay
